# 使用しているバージョン等

- ruby 2.6.5
- Rails 6.0.3.4
- MySQL

<br>

# clone
```
% git clone https://github.com/erika618/breadcrumb.git
% cd breadcrumb
% bundle install
% rails db:create
% rails db:migrate
```
